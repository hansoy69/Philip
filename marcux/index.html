Untitled Form Development and Implementation

An Educational Website on Basic Computer Skills for High School Students

General Shortcuts

  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + X: Cut
  • Ctrl + Z: Undo
  • Ctrl + Y: Redo
  • Ctrl + A: Select all
  • Ctrl + S: Save
  • Ctrl + P: Print
  • Ctrl + N: New document/window
  • Ctrl + O: Open file
  • Alt + F4: Close window or application
  • Alt + Tab: Switch between open windows
  • Windows + D: Show desktop
  • Windows + L: Lock computer
  • Windows + E: Open File Explorer

Browser Shortcuts

  • Ctrl + T: Open new tab
  • Ctrl + W: Close current tab
  • Ctrl + Shift + T: Reopen last closed tab
  • Ctrl + Tab: Switch to the next tab
  • Ctrl + Shift + Tab: Switch to the previous tab
  • Ctrl + R: Reload/refresh the page
  • Ctrl + F: Find on page
  • Alt + Left Arrow: Back
  • Alt + Right Arrow: Forward

Text Editing Shortcuts

  • Ctrl + B: Bold text
  • Ctrl + I: Italicize text
  • Ctrl + U: Underline text
  • Ctrl + Shift + L: Align text left
  • Ctrl + E: Center text
  • Ctrl + R: Align text right
  • Ctrl + K: Insert hyperlink
  • Ctrl + Shift + >: Increase font size
  • Ctrl + Shift + : Decrease font size

File Management

  • Ctrl + Shift + N: New folder
  • Ctrl + Shift + Esc: Open Task Manager
  • F2: Rename selected file
  • Ctrl + Shift + S: Save as
  • Ctrl + Alt + Del: Open Security options

System Shortcuts

  • Windows + R: Open Run dialog
  • Windows + I: Open Settings
  • Ctrl + Shift + Esc: Open Task Manager
  • Ctrl + Alt + Del: Open security options (Lock, Sign out, etc.)
